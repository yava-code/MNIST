## What's This About?

So, MNIST is this famous dataset that has like thousands of images of handwritten digits (0-9). It's basically the "Hello World" of machine learning and computer vision. Everyone starts here when they're learning about neural networks!

## What I Did

I built a neural network that can look at a picture of a handwritten digit and tell you what number it is. It's trained on 60,000 images and can actually get pretty accurate! (Way better than my own handwriting recognition skills lol)

## Files in This Repo

- `main.ipynb` - The main Jupyter notebook where all the magic happens
- `index.html` - A simple webpage showing off this project (check it out on GitHub Pages!)

## How to Run This

1. First, you'll need Python and Jupyter Notebook installed
2. Clone this repo: `git clone https://github.com/yava-code/MNIST.git`
3. Open up `main.ipynb` in Jupyter
4. Run the cells and watch the model train!
